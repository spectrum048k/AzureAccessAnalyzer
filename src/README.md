# Introduction

IAM Access Analyzer, but for Azure.

## Running locally

Set the environment variables used:

```sh
export TENANT_ID="<TENANT_ID>"
export CLIENT_ID="<CLIENT_ID>"
export CLIENT_SECRET="<CLIENT_SECRET>"
```
